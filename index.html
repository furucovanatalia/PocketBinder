Backup & Restore

Save all albums, cards, covers and adjustment data in one PocketBinder backup file.

Moving to a new device?

1. On your old device, tap Export Backup.

2. Save the JSON file to Files or iCloud Drive.

3. Open PocketBinder on your new device.

4. Go to Settings → Backup & Restore → Restore Backup and choose the saved file.

Restore replaces the albums currently stored in this browser.
